Structs

HostRef

This is used to attach the Ledger’s host structures to wasm environment, which is used for implementing some host calls. It wraps an immutable reference, so the access is thread-safe, but because of the unsafe reference conversion, care must be taken that while this reference is borrowed, no other process can modify it.

HostSlice

This is used to attach the Ledger’s host structures to wasm environment, which is used for implementing some host calls. It wraps an immutable slice, so the access is thread-safe, but because of the unsafe slice conversion, care must be taken that while this slice is borrowed, no other process can modify it.

MutHostRef

This is used to attach the Ledger’s host structures to wasm environment, which is used for implementing some host calls. Because it’s mutable, it’s not thread-safe. Also, care must be taken that while this reference is borrowed, no other process can read or modify it.

MutHostSlice

This is used to attach the Ledger’s host structures to wasm environment, which is used for implementing some host calls. It wraps an mutable slice, so the access is thread-safe, but because of the unsafe slice conversion, care must be taken that while this slice is borrowed, no other process can modify it.
